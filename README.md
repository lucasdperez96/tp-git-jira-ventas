
# Sistema de Análisis de Ventas

Proyecto académico desarrollado utilizando Git, GitHub y Jira para simular un flujo de trabajo Scrum.

## Estructura del proyecto

- datos/: contiene el dataset CSV
- scripts/: contiene el análisis en Python
- resultados/: contiene gráficos y archivos exportados

## Funcionalidades implementadas

- Cálculo de ventas totales
- Resumen estadístico
- Identificación de períodos de mayor volumen de ventas
- Generación de gráficos
- Exportación de resultados

## Ejecución

Ejecutar:

python scripts/analisis_ventas.py

## Resultados generados

- resultados/grafico_ventas.png
- resultados/resumen_ventas.csv
